          Como configurar no Railway
        

        

          
Adicione as seguintes variáveis de ambiente no painel do Railway (serviço → Variables):

          

            

              
                

                  Variável
                  Valor
                

              
              
                

                  VAPID_PUBLIC_KEY
                  Chave pública VAPID (base64url) gerada com pywebpush ou web-push-libs
                

                

                  VAPID_PRIVATE_KEY
                  Chave privada VAPID (base64url)
                

                

                  VAPID_CLAIMS_SUB
                  mailto:seuemail@dominio.com
                

              
            

          

          

            Como gerar as chaves VAPID:

            python -c "from pywebpush import Vapid; v = Vapid(); v.generate_keys(); print('PUBLIC:', v.public_key); print('PRIVATE:', v.private_key)"
            
Ou use web-push-codelab.glitch.me para gerar online.
